# AppointmentNote

โปรแกรมนี้เป็นโปรแกรมที่มีไว้บันทึกการนัดหมาย

- โปรแกรมที่ใช้เปิดไฟล์ Visual code studio 2026
- ภาษาที่ใช้ C#

## วิธีโหลด

1. เข้า github และเข้าสู่ระบบ(ถ้าไม่มีให้สมัคร)
2. เชื่อมไอดี github เข้ากับ visual code studio
3. กดปุ่ม <>code และกด open with visual studio
4. กด open visual studio

## วิธีการใช้งาน

1. เปิดโปรแกรม
2. คลิกที่ปุ่ม >Start
3. กรอกข้อมูล
4. กด Add

## คำอธิบานหน้าจอ

1. AppointmentID (รหัสนัดหมาย)
2. Subject (หัวข้อการนัด)
3. Importance (ความสำคัญ เช่น Low, Medium, High)
4. Status (เช่น Planned, Confirmed, Completed)
5. Add (สร้างการนัดหมาย)

## การเขียนที่รองรับ

โปรเจคนี้รองรับ [รายการรูปแบบไฟล์/ภาษา] ต่อไปนี้:

- C#
